# TokosDecoder — декодер и кодировщик SSTV (Python, macOS)

- **Декодирование:** из файла WAV/аудио, с выбором режима и отображением «водопада».
- **Кодирование:** изображение → WAV с настройкой размера в кадре (вписать/заполнить, масштаб, смещение).

В папке лежит **Dayton Paper.pdf** — описание режимов и таймингов (можно использовать для уточнения PD и других форматов).

## Поддерживаемые форматы

| Режим        | Описание              | VIS |
|-------------|------------------------|-----|
| Robot 36    | 320×240, YC, ~36 с    | 8   |
| Robot 72    | 320×240, YC, ~72 с    | 12  |
| Martin 1    | 320×256, GBR, ~114 с  | 44  |
| Martin 2    | 320×256, GBR, ~58 с   | 40  |
| Wraase SC2-180 | 320×256, RGB, 180 с | 55  |

Режимы **PD 50–90** (и другие PD) используют кодирование YCbCr и «2 линии на кадр» — их поддержку можно добавить по спецификации из Dayton Paper (отдельная ветка декодера).

## Требования

- Python 3.8+
- macOS (или любой ОС с Python и Tk)

## Установка

Декодер **sstv** уже лежит в папке проекта (из [colaclanth/sstv](https://github.com/colaclanth/sstv)). Нужны только зависимости:

```bash
cd tokosdecader
python3 -m venv venv
source venv/bin/activate   # Windows: venv\Scripts\activate
pip install -r requirements.txt
```

## Запуск

```bash
python3 app.py
```

**Вкладка «Декодирование»:** выберите WAV/аудиофайл, режим (Авто по VIS или конкретный), нажмите **Декодировать**. Изображение появляется по строкам («водопад»). Результат можно сохранить в PNG/JPEG.

**Вкладка «Кодирование»:** выберите изображение и режим SSTV. Настройте размер в кадре: **Вписать** / **Заполнить**, **Масштаб %**, **Смещение X/Y** — в правой части отображается предпросмотр. Нажмите **Кодировать и сохранить WAV…**.

## CLI (без GUI)

Если установлен пакет `sstv`:

```bash
sstv -d signal.wav -o result.png
```

Список режимов: `sstv --list-modes`.

## Структура проекта

- `app.py` — GUI (tkinter): вкладки «Декодирование» (файл/микрофон, выбор режима) и «Кодирование» (изображение → WAV)
- `sstv/` — декодер [colaclanth/sstv](https://github.com/colaclanth/sstv)
- `sstv_modes_ext.py` — расширение режимов (SC2-180)
- `requirements.txt` — зависимости (numpy, Pillow, PySoundFile, scipy, sounddevice, PySSTV)
